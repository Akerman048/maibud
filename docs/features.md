# BuildPhase Features

## User actions

- User can login
- User can logout
- User can see projects
- User can create project
- User can open project
- User can edit project
- User can archive project

## Document actions

- User can upload document
- User can view document
- User can update document
- User can submit document for review

## Review actions

- Reviewer can approve document
- Reviewer can reject document

## Comment actions

- User can comment on document
- User can see document comments

## Admin / organization actions

- User can create organization
- User can invite project member
- User can remove project member

## System actions

- System can save audit log
- System can show document status
